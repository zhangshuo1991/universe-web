# .impeccable.md — Universe Web Design Context

## Design Context

### Users

面向**大众科普/教育**场景的地球观测站。用户是对太空、地球科学感兴趣的普通人——可能是学生、科普爱好者、或偶然点进来的好奇访客。他们不需要专业背景就能理解界面，通过 AI 对话和交互式地球探索来学习卫星轨道、天气、地震、空间天气等知识。使用场景通常是桌面浏览器上的沉浸式探索，也需要兼顾移动端的基本可用性。

### Brand Personality

**探索 · 智能 · 优雅**

- **探索 (Exploratory)**: 界面应像太空望远镜的取景器——每一次交互都在揭示新发现。鼓励好奇心，让用户感到"再点一下就能看到更多"。
- **智能 (Intelligent)**: AI 对话和数据可视化传递"懂你想问什么"的感觉。信息呈现应精确而不堆砌，给人值得信赖的专业印象。
- **优雅 (Elegant)**: 太空的壮美不需要花哨的装饰。留白、动效、排版都应克制而考究，像一件精密仪器而非一堆仪表盘。

语调：沉稳但不冷冰冰，像一位知识丰富、善于讲故事的天文馆解说员。

### Aesthetic Direction

**视觉基调**: 深空科技感 — 纯暗色主题，玻璃态面板，微妙的光晕与网格纹理。

**参考方向** (综合三者优点):
- **NASA Eyes on the Earth**: 科技感、数据丰富、太空可视化的权威感
- **Apple 天气/地图**: 消费级精致度、极简交互、动效品质
- **Windy.com**: 数据密集但视觉直观的气象仪表盘

**反面参考** (应避免的方向):
- 过度拟物或游戏化的卡通风格
- 信息过载的传统仪表盘 (满屏数字无层次)
- 过于极客/黑客风 (纯 monospace 终端感)

**主题**: 仅暗色模式 (深空背景 `#050913`)

**色彩系统** (已建立):
| Token | 值 | 用途 |
|-------|------|------|
| `--bg` | `#050913` | 主背景 |
| `--bg-secondary` | `#091324` | 次级背景 |
| `--panel` | `rgba(8,18,32,0.9)` | 面板背景 |
| `--text` | `#edf4ff` | 主文本 (冷白) |
| `--muted` | `#96aeca` | 次级文本 (蓝灰) |
| `--aqua` | `#5eead4` | 主重点色 (青绿) |
| `--warm` | `#f7b955` | 辅助重点色 (琥珀金) |
| `--danger` | `#f87171` | 警告/错误 |

**字体组合**:
- 正文: Literata (衬线) — 传递文化感与可读性
- UI/标题: Oxanium (几何无衬线) — 传递科技感
- CJK 回退: PingFang SC → Hiragino Sans GB → Noto Sans CJK SC

**空间与形状**:
- 圆角: 18-32px (大圆角，柔和科技感)
- 间距: 基于 18px 的间距系统
- 面板: 玻璃态 (半透明 + backdrop-blur + 内发光边框)
- 布局: CSS Grid，三栏 (sidebar 104px / viewer / detail 360px)

### Design Principles

1. **数据即叙事 (Data as Story)** — 每一项数据都应有上下文和意义，而非孤立的数字。卫星数量不如"正在追踪 X 颗卫星中最亮的 ISS"有故事感。让信息自己讲述地球的故事。

2. **渐进式揭示 (Progressive Disclosure)** — 首屏保持干净壮美，细节在用户表达兴趣时展开。不要一次性铺满所有数据层。用分层、折叠、AI 对话来引导深度探索。

3. **精密但不冰冷 (Precision with Warmth)** — 像精密仪器一样可靠，但通过琥珀金色高光、柔和的圆角、优雅的动效来传递温度。科技感应该让人感到"被赋能"而非"被技术淹没"。

4. **空间感优先 (Spatial Hierarchy)** — 地球 viewer 永远是视觉焦点。面板是它的配角——半透明、不抢占注意力。信息层次靠亮度和透明度区分，而非色块和边框。

5. **无障碍 AA 基线 (WCAG AA Baseline)** — 所有文本对比度 ≥ 4.5:1 (小字) / 3:1 (大字)。交互元素有清晰的 focus 状态 (`--aqua` 轮廓)。动效尊重 `prefers-reduced-motion`。信息不仅靠颜色传达。
